PCB Order settings:
2 layers, 1mm thickness, 1oz copper

CNC parts settings:
threads: yes, meterial: aluminium 7075, surface: bead blasted
(not that important to do it exactly like this)

metal parts + PCBs should be <30€ if ordered from JLCPCB

I used small 3x10mm cylindrical neodymium magnets (only one piece needed) and yellow 3d printer hotbed springs (die cast springs) in 8mm OD, 4.5mm ID, 10mm length (~8kg each for full compression)

The hall sensor PCB needs a single 1uF 0603 capacitor and DRV5057A3EDBZRQ1 SMD hall sensor. It outputs a PWM signal that gets filtered on the QShift32 PCB.
